Kubernetes 기초

이 튜토리얼에서는 Kubernetes 클러스터 오케스트레이션 시스템의 기본 사항에 대해 설명합니다. 각 모듈에는 주요 Kubernetes 기능 및 개념에 대한 배경 정보가 포함되어 있으며 대화형 온라인 튜토리얼이 포함되어 있습니다. 이 대화형 튜토리얼을 사용하여 단순한 클러스터 및 컨테이너 화 된 응용 프로그램을 직접 관리 할 수 ​​있습니다.

대화형 튜토리얼을 이용하여 이런 것들을 배울 수 있습니다. :

  • 컨테이너화 된 앱을 클러스터에 배포 하기
  • 배포 스케일 조정 하기
  • 컨테이너화 된 앱을 새로운 소프트웨어 버전으로 업데이트하기
  • 컨테이너화된 앱 디버깅하기

이 튜토리얼에서는 Katacoda를 사용하여 웹 브라우저에서 어디서나 실행할 수있는 소규모 로컬 Kubernetes의 Minikube를 실행하는 가상 터미널을 실행합니다. 소프트웨어를 설치하거나 아무 것도 구성 할 필요가 없습니다. 각 대화형 튜토리얼은 웹 브라우저 자체에서 직접 실행됩니다.


Kubernetes 로 무엇을 할 수 있나요?

최신 웹 서비스를 사용하면 사용자는 연중 무휴로 애플리케이션을 사용할 수 있으며 개발자는 하루에도 여러 번 해당 애플리케이션의 새 버전을 배포 할 것으로 기대합니다. 컨테이너 화는 패키지 소프트웨어가 이러한 목표를 달성하는 데 도움이되어 다운 타임없이 쉽고 빠르게 응용 프로그램을 릴리스하고 업데이트 할 수 있습니다. Kubernetes는 컨테이너 화 된 응용 프로그램이 언제 어디서나 실행되도록하여 작업에 필요한 리소스와 도구를 찾을 수 있도록 도와줍니다. Kubernetes는 Production 준비가 완료된 오픈 소스 플랫폼으로, 컨테이너 오케스트레이션에 대한 Google의 축적 된 경험과 커뮤니티에서 제공하는 동종 최고의 아이디어를 결합하여 설계되었습니다.
